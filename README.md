# Selenium Java Automation Framework

This project is a Selenium automation framework built using:

- Java
- Selenium WebDriver
- TestNG
- Page Object Model (POM)
- Extent Reports
- Maven

## Project Structure

- Base – WebDriver setup
- Pages – Page Object classes
- Helpers – Utility helpers
- Utils – Common reusable utilities
- demoQATest – Test classes

## How to Run

1. Clone the repository
2. Open in Eclipse or IntelliJ
3. Run DemoQATest.java as TestNG test

## Reporting

Extent Report is generated inside:
test-output/ExtentReport.html
